# Personal Portfolio

This project is my personal portfolio website showcasing my work in **Front-End Development** and **UX Design**.

The goal of this portfolio is to present a selection of projects that demonstrate my skills in building responsive websites, writing clean code, and designing user-centered digital experiences.

## Projects Included

### Front-End Projects

- **Rainy Days**  
  A front-end e-commerce website focused on selling rain jackets. The project demonstrates responsive design and JavaScript functionality.

- **Community Science Museum**  
  A responsive website designed for a fictional science museum. The project focuses on layout structure, accessibility, and usability.

- **Studio Hybrid**  
  A blog-style website developed as part of an exam project. The goal was to create a dynamic and responsive platform for exploring posts.

### UX Design Projects

- **Digital Nomad**  
  A mobile app concept designed to help freelancers gain visibility and create new opportunities by showcasing their work and offering courses.

- **Airbnb UX Improvement**  
  A UX project focused on improving the navigation and booking experience on the Airbnb platform through research and usability testing.

- **Okalpha Contact Form**  
  A high-fidelity prototype designed to improve accessibility and usability for a contact form page.
